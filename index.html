Game Project
No.1
Colliball
· Introduction
  This is a casual game which is simple, funny and fresh in image. It’s suitable to be entertained with friends together before or after meals to kill the time.

· Duty
  Software Development

· Advisor
  Prof. Qiang Liu, Tsinghua Univ.

· Dowload Link
  Click Here.

· How to Play & Demo
· Intention
  Because I like using solid-colored fresh image style, and I am learning HTML and JavaScript, I would like to develop a game with such character as Web homework. Inspired by OLO(a mobile game), my classmate and I decided to transform it into web game to experience the development of a complete gaming.

· My Personal Contribution
  This is a group project that I cooperate with another classmate. I am responsible for the mode writing of 2nd players and 4th players, and my classmate is responsible for writing the on-line version of the game. (The on-line version of the game is not available in the content in the link)

· What I’ve Learned
  During writing Colliball, I fully learned how to write a game-related Web page (HTML, CSS, js), mastered the usage of game engine (an open source 2D engine called LufyLegend is applied in the process of writing). I learned to communicate with the developers because during the writing I have mailed the engine developer and game developer. I am indebted to those developers who gave me a lot of help.

· Platform Requirement
  Any platforms which can get access to network (Windows, Mac) and on which mobile phones can run.

· How To Install
  Just need to unzip and open the file (index.html) then you can start the game.

· Related File
  Index.html ---Game main interface
  Css ---Web page formatting code
  Img ---Related picture files
  Js ---Related game script files
  Sound ---Game sound effects and background music

No.2
FruitRun
· Introduction
  FruitRun is a fruit themed parkour game. The protagonist is a mangosteen rolling in the pipe, and it needs to collect the coins on the road while dodging flying knives and needles on the ground, breaking through closed doors and other obstacles.

· Duty
  Software Development

· Advisor
  Prof. Qiang Liu, Tsinghua Univ.

· Dowload Link
  Click Here.

· How to Play & Demo
· Intention
  I want to learn to use the Unity3D engine, and accumulate experience of designing and writing games. By chance my second degree (digital entertainment design) classmates wanted to develop a game, so we decided to build a team, to experience the development of a complete gaming.

· My Personal Contribution
  Our team consists of four members. All staff are involved in the game design discussions. One of us is responsible for the visual design. Another and I are responsible for the game script writing.

· What I’ve Learned
  In the very beginning of investigation about Parkour game-related content, I learned a lot of valuable experience and perspective from other developers (students and similar games developers). In the process of designing playing method, I learned to express my ideas to others, and also gained lots of highlights from other people's thoughts and ideas. I learned how to use the Unity3D engine and master the c # game script writing. We needed to take the game to attend TENCENT MINIGAME College Students' Creative Competition which there was a deadline, so I learned how to arrange a reasonable game development schedule, which kept us from being hurried before deadline.

· Platform Requirement
  Android

· How To Install
  Install APK files on the Android platform

· Related File
  FruitRunV2.5.apk ---Game Installpack
